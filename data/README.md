# 📊 Data Description

The dataset used in this study is publicly available at:

**Mendeley Data (DOI):**
https://doi.org/10.17632/2tpv28kr83.3

## 📁 Contents in Mendeley Repository

* Raw Data
* Final Dataset used in the study
* Exploratory Data Analysis
* Evaluation Metrics Comparison
* Chaotic Analysis (False Nearest Neighbors)
* Chaotic Model (Echo State Network - ESN)
* Machine Learning Models (FFNN, ELM)
* Deep Learning Models (LSTM, GRU)
* Ensemble Learning Models (RF, XGBoost)
* Stacking Ensemble Models

## ⚠️ Note

Due to size and reproducibility considerations, the dataset is not stored in this GitHub repository.

## ▶️ How to Use

1. Download the dataset from the link above
2. Place the relevant files inside this `/data` folder
3. Ensure file paths match those used in the scripts
